## Installation

1. Clone this repository:  
   ```bash
   git clone https://github.com/username/portfolio-website.git
    ````
2. Navigate to the project directory: 
    ```bash
    cd portfolio-website
    ````

3. Install dependencies:
    ```bash
    npm install
    ````

4. Start the development server:
    ```bash
    npm run start
    ````
